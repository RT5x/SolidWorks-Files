If there are issue that prevent the uninstall:
Try a complete uninstallation of SOLIDWORKS using the Microsoft fixit tool.

Download the Microsoft fixit tool from this link:
https://support.microsoft.com/en-us/help/17588/fix-problems-that-block-programs-from-being-installed-or-removed

Run the downloaded tool and click Next
Click Uninstalling
In product selection select ' SOLIDWORKS (version) ' and click Next to follow uninstallation process
Repeat the process until all the versions of SOLIDWORKS and all the SOLIDWORKS products are uninstalled
